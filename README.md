# Circus Smash — Smash Burgers (Black / White / Red)

Landing page for Circus Smash, full menu, hours, and both locations, in a black/white/red checkered cartoon style with an accordion-style menu.

## Sections
- Hero with checkerboard background, stat badges, rotating circle badge
- Accordion menu: Burgers (with pricing table + pollo/veggie cards), Para Picar, Postres, Bebidas, plus Extras & Salsas
- Both locations with addresses, phone numbers, opening hours
- Review cards with avatar initials + summary score
- Order CTAs (Uber Eats link + internal anchor)

## Files
```
index.html         Main page
assets/style.css    Styles (black/white/red, checkered cartoon)
assets/script.js    Accordion menu behavior
index_standalone.html   Self-contained single-file version
```

## Deploying
Plain HTML/CSS/JS, no build step. Works on GitHub Pages, Vercel, Netlify — `index.html` at root.

## Note
Menu items, prices, hours, addresses, and the Uber Eats link were provided directly — confirm against the restaurant's current listings before publishing.
